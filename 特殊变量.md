## $event

触发事件时，可用 `$event` 捕获当前事件对象，如下：

```html
<button on-click="submit($event)">
    Submit
</button>
```

> 调用方法如果没有参数，默认会把事件对象传进来，因此这里写与不写 `$event` 是一样的。
>
> 加上这个特性主要是方便多个参数时修改 event 参数的位置。

## $keypath

在模板的任何位置，可用 `$keypath` 获取当前 keypath，如下：

```html
{{#each array}}
    {{$keypath}}
{{/each}}
```
