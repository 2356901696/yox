`if` 或 `else if` 后面跟一个表达式，语法和 js 完全一致。

> Mustache 的原版语法，空数组被认为是 `false`，这个设计严重违背工程师的第一直觉，因此我们把它剔除了。

```
{{#if expression}}
    ...
{{else if expression}}
    ...
{{else}}
    ...
{{/if}}
```

如果对 `else if` 这种带空格的默认配置不满意，可以自行修改配置，比如 `{{elseif expression}}` 或者 `{{elif expression}}`。

> 语法配置请参考“**全局配置**”章节
