监听数据变化有两种方式：

* 配置 watchers
* 手动调用 watch(keypath, watcher)

不论使用哪种方式，watcher 函数的参数列表是一样的，都是 `[newValue, oldValue]`。

## 配置

```javascript
{
    watchers: {
        name: function (newValue, oldValue) {

        }
    }
}
```

## 手动调用

```javascript

{
    onattach: function () {
        this.watch('name', function (newValue, oldValue) {

        });
    }
}
```
