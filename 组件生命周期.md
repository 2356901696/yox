组件的生命周期顺序如下：

```
init -> create -> compile -> attach -> update -> detach
```

## init

进入 `new Yox(options)` 之后立即触发，钩子函数会传入 `options` 。

## create

除了还没编译模板，其他该做的都做完了。

## compile

模板编译结束后触发。

## attach

组件加入 DOM 树后触发。

这时可通过 `$el` 获取组件根元素。

## update

组件视图更新后触发。

## detach

组件从 DOM 树移除之前触发。

如果在 `attach` 时初始化过第三方 DOM 库，切记在 `detach` 时销毁。
