## 事件



组件内部的事件，默认不冒泡，这意味着事件不会传递到组件外部。



如果想监听组件内部事件，可通过事件指令实现，如下：



```html

<Component on-xxx="yyy" />

```



```javascript

{

 yyy: function () {

 // 处理 yyy 事件

 }

}

```



如果组件内部触发的是冒泡事件，那么在目标组件的父级组件中，能自动监听，如下：



```javascript

// 目标组件触发一个冒泡事件

component.fire('xxx', true)

```



```javascript

// 父级组件

{

 events: {

 xxx: function () {

 // 处理 xxx 事件

 }

 }

}

```
