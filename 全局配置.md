## 开启 debug 模式

York 日志消息分为两种，一种是提示，一种是错误。

对于提示，只有在 debug 模式下才会打印出来。

```javascript
York.config.debug = true;
```

## 开启异步更新

默认是同步更新。

如果想实现 Vue 那样的异步更新，可按如下配置：

```javascript
York.config.sync = false;
```

在视图更新后做一些事，可调用 `York.nextTick` 方法：

```javascript
York.nextTick(function () {
    // 此时视图已更新
    // balabala
});
```

```javascript

```

## 注册全局过滤器

