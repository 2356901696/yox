## 是否启用异步更新

默认是同步更新，如果想像 Vue 那样的异步更新，可按如下配置：

```javascript
York.config.sync = false;
```

```javascript
York.nextTick(function () {
    // 此时视图已更新
    // balabala
});
```

## 注册全局组件

```javascript
York.component('name', {
    template: 'xx',
    data: function () {
        return {

        };
    },
    ...
});
```

## 注册全局指令

```javascript
York.directive('name', {
    attach: function () {

    },
    update: function () {

    },
    detach: function () {

    }
});
```

## 注册全局子模板

```javascript
York.partial('name', 'template');
```

## 注册全局过滤器

```javascript
York.filter('name', function (value) {
    return value;
});
```
