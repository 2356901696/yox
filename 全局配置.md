## 注册全局组件

```javascript
York.component('name', {
    template: 'xx',
    data: function () {
        return {

        };
    },
    ...
});
```

## 注册全局指令

```javascript
York.directive('name', {
    attach: function () {

    },
    update: function () {

    },
    detach: function () {

    }
});
```

## 注册全局子模板

```javascript
York.partial('name', 'template');
```

## 注册全局过滤器

```javascript
York.filter('name', function (value) {
    return value;
});
```
